# Как запустить

- Закинуть .ipynb на гугл диск
  
- Распаковать archive.zip, закинуть папку AirPollutionSeoul на гугл диск рядом с .ipynb

- Распаковать maps-of-seoul.zip, закинуть папку maps-of-seoul на гугл диск рядом с .ipynb
